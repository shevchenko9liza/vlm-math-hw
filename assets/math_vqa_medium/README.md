# math_vqa_medium

Средний синтетический public-набор для VLM Math Homework.

Содержит 260 примеров:

- train: 180
- dev: 40
- test_public: 40

Темы:

- algebra: line plots y=ax+b
- plots: bar charts
- geometry: grids, right triangles, adjacent angles
- linear_algebra: 2x2 determinant
- coordinate_geometry: squared distance on coordinate plane

Формат совпадает с `assets/toy_math_vqa/manifest.jsonl`. Набор полностью синтетический и не требует скачивания внешних датасетов.
